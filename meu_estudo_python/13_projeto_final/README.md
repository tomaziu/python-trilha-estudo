# 13 — Projeto Final

Projeto integrador: Gerenciador de Contatos com CSV, pandas, OOP e tratamento de erros.

## Funcionalidades

1. Cadastrar contato (nome, telefone, email, cidade)
2. Listar todos os contatos
3. Buscar por nome
4. Filtrar por cidade
5. Salver e ler do arquivo contatos.csv
6. Mostrar estatísticas (total por cidade)

## Arquivos

- `main.py` — menu e fluxo principal
- `contato.py` — classe Contato
- `contatos.csv` — dados salvos
