# Trilha de Estudo — Python 🐍

Repositório com meu progresso de aprendizado em Python, do básico ao avançado.
Aqui você encontra exercícios práticos, anotações, projetos e desafios organizados por tópico.

> Este projeto é baseado no meu próprio estudo e evolução na linguagem. Cada exercício foi desenvolvido durante minha jornada de aprendizado, com o objetivo de consolidar os conceitos de forma prática.

---

## Pré-requisitos

- Python 3.x instalado → [python.org/downloads](https://www.python.org/downloads/)
- Noções básicas de terminal/linha de comando
- Editor de código (recomendo VS Code)

---

## Como usar este repositório

As pastas dentro de `meu_estudo_python/` estão numeradas e devem ser seguidas **em ordem**.  
Cada pasta contém:
- Anotações e explicações do tópico
- Exercícios práticos para fixar o conteúdo
- Exemplos de código comentados

Siga do `01_fundamentos/` até o `13_projeto_final/` para uma trilha completa do básico ao avançado.

---

```
meu_estudo_python/

```

---

## Como instalar

```bash
# Clonar o repositório
git clone https://github.com/tomaziu/python-trilha-estudo.git
cd python-trilha-estudo

# Criar ambiente virtual
python -m venv venv

# Ativar (Windows)
venv\Scripts\Activate.ps1

# Ativar (Linux/Mac)
source venv/bin/activate

# Instalar dependências
pip install -r requirements.txt
```
